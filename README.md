# Graphics
Explores more advanced rendering techniques.

# Setup
Created a base project using template by copying over https://docs.vulkan.org/tutorial/latest/00_Introduction.html

1. Setup the environment using https://docs.vulkan.org/tutorial/latest/02_Development_environment.html
2. Create build folder
3. Run `cmake ..`
4. Run `make`
5. Run program inside `Graphics/`
